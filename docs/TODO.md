# 目的
手機桌布截圖器的主要用途是為了解決使用者使用手機桌布切換器時，  
軟體會自動將比例置中擷取為手機螢幕，但很多圖片的焦點並非位於中間，  
直接擷取會使圖片的美感降低。
但截圖功能只是圖片編輯的其中一部分，因此類似PS等軟體在處理此事務時顯得臃腫而麻煩。  
想開發一個更快處理這類事務的桌面軟體。

## 需要功能 ⮇
- [ ] 讀取並載入圖片預覽
  - 需要解碼器
  - 調整為適合視窗
- [ ] 建立視覺明顯的擷取框
  - 跟圖片明度相反的邊框?
  - 擷取框外以半透明灰色覆蓋(適合明度?)
- [ ] 依據手機型號選擇擷取框比例
  - 硬編碼主流機型
  - 某種api查詢?
- [ ] 以手機螢幕解析度儲存圖片
  - 原格式或webp
- [ ] 根據不同快捷鍵搭配以不同程度的移動擷取框
  - 圖片移動而非擷取框移動
  - 方向鍵 1px、+ctrl 1%、+shift 10%?
- [ ] 將擷取框位置儲存json以多次編輯同一圖片
  - 微調需求?
- [ ] 以手機螢幕大小預覽圖片
- [ ] 擷取框可橫置
  - 為了平板或筆電?
